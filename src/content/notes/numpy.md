# NumPy 基础

## NumPy 是什么

- 是一个开源的 Python 科学计算库
- 使用 NumPy 可以方便地使用数组、矩阵进行计算
- 包含线性代数、傅里叶变换、随机数生成等大量函数

## NumPy 的优势

- **代码更加简洁**
  - NumPy 直接以数组、矩阵为粒度计算并且支持大量的数学函数，而 Python 需要用 for 循环从底层实现。
- **性能更加高效**
  - NumPy 的大部分代码是用 C 语言实现的，所以比纯 Python 要高效
