# Lesedi's Notes

个人技术笔记博客 — 前端、后端、AI、面试经验。基于 Astro 构建，自动部署到 GitHub Pages。

**在线访问：** https://lxxxxx-oss.github.io/Lesedi-blog/

## 本地运行

```bash
npm install
npm run dev        # → http://localhost:4321
npm run build      # → dist/
```

## 技术栈

Astro 6 + Markdown + CSS（Bear Blog 风格），无 UI 框架依赖。
